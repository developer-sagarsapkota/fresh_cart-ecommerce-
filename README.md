FreshCart is a minimum viable product (MVP) e-commerce application built with Django and React. It provides the basic features needed for an online store with a clean and simple setup.

Features 
- User authentication (signup, login, logout)
- Product listing and detail pages
- Shopping cart (add, update, remove items)
- Checkout and order creation
- Responsive React frontend connected to Django REST API

Future improvements may include:
- Payment integration
- Product search and filters
- Admin dashboard for inventory/orders
- Docker deployment
